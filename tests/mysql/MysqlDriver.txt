
This test runs only with a valid Qt MySQL driver in place. It is tested with QT 5.15.2

Requirements to build for Windows
---------------------------------

A. Download and install the MySQL Installer (tested with 8.0.26)
B. Make sure you have the qt sources available/installed in %QTDIR%
C. Visual Studio >= 2019 (matching your qt version)

Steps to build
--------------

1. Launch the Visual Studio command prompt

2. Add the Qt binary path to PATH. 
2.a qmake --version should be avail

3. Change directory to the build path
cd %TEMP%
mkdir build-qmysql
cd build-qmysql

4. Create the makefiles with qmake
qmake -- MYSQL_INCDIR="%MYSQL%\MySQL Server 8.0\include" MYSQL_LIBDIR="%MYSQL%\MySQL Server 8.0\lib" %QTDIR%\src\plugins\sqldrivers\mysql\mysql.pro

5. Run the makefile 
nmake 

6. Install 
nmake install

7. Add the library and include to the enviroment
7.a QtCreator: Add "%MYSQL%\MySQL Server 8.0\lib", "%MYSQL%\MySQL Server 8.0\bin" and "%QTDIR%\src\plugins\sqldrivers\plugins\sqldrivers" to the path of the build enviroment 
8. The mysql test should pass
